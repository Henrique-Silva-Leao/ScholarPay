📌 Título do Projeto
Desenvolvimento de um Sistema Web de Moeda Digital Institucional para Bonificação e Transferência entre Alunos

📖 Resumo
Este projeto tem como objetivo o desenvolvimento de um sistema de carteira digital acadêmica com moeda institucional fictícia, voltado ao incentivo e à valorização do engajamento estudantil. A proposta consiste na criação de uma plataforma digital na qual a instituição de ensino poderá conceder bonificações aos alunos por meio de créditos virtuais, que poderão ser movimentados internamente através de transferências entre usuários.

O sistema contempla funcionalidades como autenticação segura, controle de saldo, registro detalhado de transações e painel administrativo para gerenciamento das bonificações. A solução busca aplicar conceitos de modelagem de sistemas, banco de dados relacional, segurança da informação e desenvolvimento de software, conforme as boas práticas da área de Análise e Desenvolvimento de Sistemas.
